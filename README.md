# Workbench Survey Editor — Prototypes

Interactive HTML prototypes for the Workbench Survey Editor project.  
Each prototype is self-contained — open the `index.html` in any browser to view.

## Prototypes

| Prototype | Description | Status |
|-----------|-------------|--------|
| [keyboard-shortcuts](./keyboard-shortcuts/) | Keyboard shortcuts with toast feedback, onboarding tour, and shortcuts modal | ✅ Ready for review |
| [vh-prototype](./design-prototypes/vh-prototype.html) | Interactive VH survey-editor HTML prototype (self-contained). | ✅ Ready for review |

## Design System

All prototypes follow **MMDS (Macromill Design System)** tokens and the staff theme.

## How to View

**Locally:** Download and open any `index.html` in your browser, or open `design-prototypes/vh-prototype.html`.  
**Online:** Enable GitHub Pages (Settings → Pages → deploy from branch `main`, folder `/ (root)`), then open:  
`https://mm-afia.github.io/workbenchse-prototype/design-prototypes/vh-prototype.html`
